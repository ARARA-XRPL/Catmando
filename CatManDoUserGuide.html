CatManDo User Guide

 

What CatManDo does and doesn't :

CatMando is an Interface dedicated to the catalog manager for managing Catalogue of Service in a Web architecture. You load it into Mozilla Firefox browser. As this tool has been developed for English language users paid attention on the default language of your browser and check that it has been put in English.

CatManDo is not a Web site it is a collaborative application set up with Web technology. It runs with a WAMP server and is written with HTML, javaScript, PHP MySql.

The look and feel is like a control panel that allowed the management of the catalog data base.

A range of buttons is designed horizontally for general purposes such as login / logout. And another range of button is put in a vertical block for business purposes like data base creation or data base controling.

An exchange area is given to manage error messages or user information and output.

Send Catalogue to Web Server

Asuming that you have a catalogue of service somewhere in the directories of your local computer.

Catmando allows you to browse your file system in order to select the catalogue file you want to record on the Web Server.

You may archive all your catalogues of service and a file log is set up on the server where all your files names will be saved for a futur usage.
